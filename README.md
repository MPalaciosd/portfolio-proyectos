# 🛠️ Portfolio de Proyectos Técnicos — Manuel Palacios

Proyectos desarrollados con IA (Claude) aplicando tecnologías modernas de backend, frontend, automatización e inteligencia artificial.

---

## 📁 Proyectos

### 1. 🤖 [popbot-agente-ia](./popbot-agente-ia)
**Agente IA de gestión de citas con monetización**

Bot conversacional inteligente para gestión de citas, construido con arquitectura de producción lista para escalar.

- **Stack:** Node.js · Express · MongoDB · Groq (LLM) · Socket.io · Stripe · JWT
- **Características:** Autenticación, pagos, disponibilidad, emails, WebSockets en tiempo real
- **Deploy:** Render.com

---

### 2. 📊 [registro-ventas-react](./registro-ventas-react)
**App de registro y gestión de ventas**

Aplicación web moderna para gestión de ventas con base de datos en tiempo real.

- **Stack:** React 19 · Firebase · Tailwind CSS
- **Características:** CRUD de ventas, interfaz responsive, sincronización en tiempo real

---

### 3. 🧠 [jarvis-asistente-voz](./jarvis-asistente-voz)
**Asistente de voz personal estilo Jarvis**

Asistente de escritorio activado por voz, inspirado en el asistente de Iron Man.

- **Stack:** Python · SpeechRecognition · pyttsx3
- **Características:** Reconocimiento de voz, respuestas por audio, comandos personalizados

---

### 4. 🎬 [movidic-webapp](./movidic-webapp)
**Web app de gestión de contenido multimedia**

Aplicación web para exploración y gestión de contenido audiovisual.

- **Stack:** Python · HTML/CSS/JS
- **Características:** Interfaz web ligera, integración con APIs de contenido

---

### 5. 👥 [crm-gestion-clientes](./crm-gestion-clientes)
**CRM completo con IA integrada**

Sistema de gestión de relaciones con clientes con módulos de seguimiento, presupuestos y comunicación.

- **Características:** Pipeline de ventas, seguimiento de clientes, presupuestos automatizados, demos IA

### 6. 📄 [clasificador-pdfs-ia](./clasificador-pdfs-ia)
**Clasificador automático de PDFs con IA**

Herramienta para procesar y clasificar documentos PDF usando inteligencia artificial.

- **Características:** Extracción de texto, clasificación automática, agrupación de incidencias

### 7. ⚙️ [playwright-tools-automatizacion](./playwright-tools-automatizacion)

### 9. 🌐 [repogit-bot-web](./repogit-bot-web)
